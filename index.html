Introduction

Gesture-Control est une application innovante qui vous permet de contrôler votre ordinateur avec des gestes de la main ou des mouvements du nez, capturés via votre webcam. Intuitive et accessible, elle offre une nouvelle façon d'interagir avec votre appareil.

Installation

Prérequis

  • Python 3.7 ou supérieur
  • Une webcam

Bibliothèques Requises

  • opencv-python
  • mediapipe
  • pyautogui
  • PyQt5
  • numpy

Étapes d'Installation

  1. Clonez le dépôt : 
    git clone https://github.com/votreusername/gesture-control.git
cd gesture-control
  2. Créez un environnement virtuel : 
    python -m venv venv
source venv/bin/activate  # Sur Windows : venv\Scripts\activate
  3. Installez les dépendances : 
    pip install -r requirements.txt
  4. Lancez l'application : 
    python main.py

Utilisation

Voici comment utiliser Gesture-Control :

  1. Démarrage : Lancez le programme avec la commande ci-dessus.
  2. Calibrage : Suivez les instructions pour calibrer l'application avec votre webcam.
  3. Gestes : Utilisez les gestes définis pour contrôler le curseur, cliquer, etc.

Pour plus de détails sur les gestes, consultez la section Gestes.

Gestes

Voici une liste des gestes supportés :

  • Déplacer le curseur : Levez l'index et bougez la main.
  • Clic gauche : Pincez le pouce et l'index.
  • Clic droit : Pincez le pouce et le majeur.
  • Défilement : Levez l'auriculaire et bougez la main verticalement.
